# Hotel Management System

Simple console-based Java app to manage hotel guests and rooms.

Requirements: Java JDK 8+

Quick start (PowerShell):

```powershell
# Open PowerShell and change to the project directory, e.g.:
cd path\to\your\project

# Compile all Java files in the project root
javac *.java

# Run the application (Main is the entry point)
java Main
```
